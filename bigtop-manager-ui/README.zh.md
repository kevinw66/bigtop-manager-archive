# Bigtop-Manager-ui

Bigtop-Manager-ui是Manager平台的前端UI，存放有关平台与用户交互，界面展示，控件样式等代码。

## 先决条件

Vite：版本4.4.5

Typescript：版本5.0.2

Vue：版本3.3.4

编辑器：VsCode

## 项目结构

```
——————————————————public           公共静态资源
——————————————————src              项目源代码
———————————api                     调用后端接口
———————————assets                  静态资源
———————————components              自定义组件
———————————layouts                 Vue页面
———————————locales                 国际化
———————————pages                   页面组件
———————————router                  路由
———————————store                   全局持久化
———————————types                   数据类型
———————————utils                   工具函数
———————————App.vue                 项目根组件
———————————main.ts                 项目打包入口
——————————————————index.html       项目页面
——————————————————package.json     项目的包管理配置文件
——————————————————vite.config.ts   项目配置文件
```
